Flex Grow

Esta regra permite alterar a proporção de um ou mais elementos filhos. Para isso, a width precisa estar sem valor, ou seja, indefinida.

Um elemento com grow igual a 2, o elemento vai crescer duas vezes mais do que os outros elementos ao se adaptar no conteiner.
